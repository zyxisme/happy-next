使用以下命令启动 Gemini 会话：

```bash
happy gemini
```

## 前置条件

安装 [Gemini CLI](https://github.com/google-gemini/gemini-cli)。

通过 Happy 连接你的 Google 凭据：

```bash
happy connect gemini
```

## 模型配置

设置默认的 Gemini 模型：

```bash
happy gemini model set gemini-3.5-pro-preview
happy gemini model get    # 显示当前模型
```

可用模型涵盖 Google 当前的 Gemini 阵容，分为 **Pro**（最强大）、**Flash**（快速的前沿模型）和 **Flash-Lite**（轻量、针对成本优化）几个档位。运行 `happy gemini model get` 可查看当前模型，或在移动应用中浏览最新列表。

## Google Workspace 账户

如果你使用 Google Workspace 账户，需要设置 Cloud Project ID：

```bash
happy gemini project set my-project-id
happy gemini project get    # 显示当前项目
```

## 工作原理

Happy 使用 Agent Client Protocol (ACP) 启动 Gemini CLI 并通过该协议进行通信。所有 Happy 功能（移动端控制、语音、编排）均适用于 Gemini 会话。
