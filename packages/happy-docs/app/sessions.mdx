The sessions screen is the heart of the Happy app. It shows all your active and recent sessions across all your machines.

## Viewing sessions

The home screen shows your sessions, with active sessions at the top. Each session card shows:

- The agent type (Claude, Codex, or Gemini)
- The model being used
- The working directory
- A preview of the latest message
- An unread indicator for new messages

Tap a session to open it.

## Sending messages

In a session, type a message at the bottom of the screen and send it. The message is delivered to the AI agent running on your machine. You'll see the agent's response in real time.

You can also:
- **Attach images** — Tap the image icon to send screenshots or photos, with or without accompanying text
- **Paste from clipboard** — Long-press the input field to paste

## Creating a new session

Tap the **+** button on the home screen — or the new-session button in an open session's header — to create a new session. You can configure:

- **Machine** — Which computer to run the session on
- **Agent** — Claude, Codex, or Gemini
- **Model** — The specific model to use
- **Working directory** — The project folder on your machine
- **Permission mode** — How the agent handles permission requests

The initial message is optional — you can create the session without typing anything and start interacting once it's running.

<Info>
Creating a session from the app requires the daemon to be running on the target machine. See [Daemon](/cli/daemon).
</Info>

## Session actions

Long-press a session card or use the session menu for additional actions:

- **Duplicate** — Create a copy of the session with the same configuration
- **Fork** — Start a new session from a specific point in the conversation
- **Resume** — Resume an ended session
- **Delete** — Remove the session permanently

## Pending queue

If the AI agent is busy processing a previous message, your new messages are queued. The pending queue shows queued messages with their status. Messages are dispatched automatically when the agent is ready.
