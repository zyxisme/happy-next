---

Start a Gemini session with:

```bash
happy gemini
```

## Prerequisites

Install the [Gemini CLI](https://github.com/google-gemini/gemini-cli).

Connect your Google credentials through Happy:

```bash
happy connect gemini
```

## Model configuration

Set the default Gemini model:

```bash
happy gemini model set gemini-3.5-pro-preview
happy gemini model get    # Show current model
```

Available models span Google's current Gemini lineup across **Pro** (most capable), **Flash** (fast frontier model), and **Flash-Lite** (lightweight, cost-optimized) tiers. Run `happy gemini model get` to see the active model, or browse the up-to-date list in the mobile app.

## Google Workspace accounts

If you use a Google Workspace account, you need to set your Cloud Project ID:

```bash
happy gemini project set my-project-id
happy gemini project get    # Show current project
```

## How it works

Happy spawns Gemini CLI with the Agent Client Protocol (ACP) and communicates through it. All Happy features (mobile control, voice, orchestration) work with Gemini sessions.
